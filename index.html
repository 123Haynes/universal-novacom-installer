Welcome to the Universal Novacom Installer.

If you're searching for older Versions (<1.4) please take a look at the original site: http://code.google.com/p/universal-novacom-installer/

Dependencies

You need at least Java >1.5 to use this installer. If you don't have it installed already, head over to Java.com and download the JRE. In case you want to use this installer with Linux, also make sure the following packages are installed:

  • xterm
  • dpkg
